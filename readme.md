# SAW-Banyu
 
1. Create new environtment 
```bash
python -m venv myenv
```

2. Activate your environment
use this line from your root project
```bash
.\myenv\Scripts\Activate
```

3. Install depedency
```bash
pip install -r requirements.txt
```

4. Run your project
```bash
python main.py
```

5. Enjoy
